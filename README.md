# `v-limine`

A V library for handling Limine boot protocol structures.

## Install

You can install the package via:

```bash
v install https://github.com/wenxuanjun/v-limine
```

## Resources

- [Limine Boot Protocol Specification](https://github.com/limine-bootloader/limine-protocol/blob/trunk/PROTOCOL.md)

## Barebones

An example project utilizing the crate can be found [here](https://github.com/plos-clan/limine-v-template).
